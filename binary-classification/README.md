# Binary Classification

## Instructions to build project

1. `git clone https://github.com/soumik12345/Youtube-DL`

2. `cd binary-classification`

3. `python3 -m pip install -r requirements.txt`

4. `chmod +x ./run.sh`

5. `./run.sh`
